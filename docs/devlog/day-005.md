# Day 005: Pseudo-Legal Pawn Move Generation

## Summary

Added pseudo-legal pawn move generation for the side to move, plus a CLI command for inspecting generated pawn moves from a FEN.

## Supported Pawn Rules

The pawn generator now supports:

- single pushes to empty squares
- double pushes from the starting rank when both path squares are empty
- diagonal captures of opposing pieces
- quiet promotions to queen, rook, bishop, and knight
- promotion captures to queen, rook, bishop, and knight
- edge-file handling so pawns do not wrap around the board

Generation uses the current `Position` and only emits moves for `position.side_to_move()`.

## Not Supported Yet

This is pseudo-legal pawn generation only. It does not support:

- en passant
- check detection
- pinned-piece restrictions
- king safety
- move making or unmaking
- non-pawn move generation
- full legal move generation
- real perft

## CLI Debug Command

Inspect generated pawn moves from a FEN:

```bash
cargo run -- movegen pawns --fen startpos
```

`startpos` is a shortcut for the standard starting FEN.

Example output:

```text
a2a3
a2a4
...
total: 16
```

## Local Checks

`make check` now includes:

```bash
cargo run -- movegen pawns --fen startpos
```
