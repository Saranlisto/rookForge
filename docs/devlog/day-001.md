# Day 001: Repository Scaffold

## Summary

Created the first Rookforge repository scaffold. The project now has a Rust workspace, core engine module boundaries, a small CLI binary, initial tests, CI, and basic documentation.

## What Exists

- Workspace root `Cargo.toml`
- `rookforge-core` library crate
- `rookforge-cli` binary crate
- Basic CLI support for:
  - `rookforge --version`
  - `rookforge help`
  - `rookforge perft --help`
- Placeholder types:
  - `Color`
  - `PieceKind`
  - `Piece`
  - `Square`
  - `Move`
- Unit and CLI tests
- GitHub Actions workflow for Rust formatting, tests, and builds

## What Does Not Exist Yet

- Board state
- FEN parsing
- Move legality
- Perft execution
- Search
- Evaluation
- UCI protocol handling

## Next Step

Add board state and FEN parsing with focused tests for valid and invalid positions.
