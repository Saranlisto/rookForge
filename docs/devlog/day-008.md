# Day 008: Combined Pseudo-Legal Move Generation

## Summary

Added a combined pseudo-legal move generation API that gathers all currently implemented piece-specific generators into one deterministic move list.

## Combined Move Generation

`generate_pseudo_legal_moves` now includes:

- pawns
- knights
- bishops
- rooks
- queens
- kings

The generator only produces moves for the side to move. For now, output is deterministic by piece family: pawns, knights, bishops, rooks, queens, then kings.

## Starting Position

The standard starting position now produces 20 pseudo-legal moves:

- 16 pawn moves
- 4 knight moves

Sliding pieces and kings remain blocked in the starting position.

## CLI Usage

```bash
cargo run -- movegen all --fen startpos
cargo run -- movegen all --fen "8/8/8/3Q4/8/8/4P3/4K3 w - - 0 1"
```

Each command prints normalized UCI moves followed by a total count.

## Local Checks

`make check` now includes:

```bash
cargo run -- movegen all --fen startpos
```

This keeps the local smoke workflow aligned with the new combined move generator.
